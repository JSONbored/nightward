# Code of Conduct

Nightward uses the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Standards

Keep discussion technical, respectful, and focused on making the project safer and more useful. Do not post secrets, private config, credential paths, private support bundles, or unredacted local agent state.

Examples of expected behavior:

- Use redacted fixtures when reporting parser or adapter issues.
- Explain risk and tradeoffs concretely.
- Respect maintainers' decisions around no-telemetry, no-network-runtime, and no-mutation-by-default boundaries.

Examples of unacceptable behavior:

- Harassment or personal attacks.
- Publishing another person's secrets, config files, or private paths.
- Pressuring maintainers to add unsafe default writes or secret-copying behavior.

## Enforcement

Maintainers may edit, hide, or remove issues, comments, pull requests, or other contributions that violate these standards. Serious or repeated violations may lead to a ban from project spaces.

## Reporting

For conduct issues, open a GitHub issue only if it can be discussed publicly. For security or private-state exposure, use the process in [SECURITY.md](SECURITY.md).
