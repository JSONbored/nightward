# Raycast

Nightward's Raycast extension is a read-only macOS companion.

## Commands

- Dashboard.
- Findings.
- Analysis.
- Provider Doctor.
- Explain Finding.
- Explain Signal.
- Export Fix Plan.
- Export Analysis.
- Open Reports.

The extension shells out to `nw` or `nightward`, renders redacted output, and never mutates agent configs.
